# EtharaChats — Production-Ready Setup Guide

EtharaChats is a real-time, ultra-lightweight group chat web application. It is built completely framework-free on the frontend using semantic HTML5, clean modern CSS3 (featuring a rich dark palette with orange and pink accents), and vanilla browser JavaScript. The backend is powered by a native Node.js HTTP server configured with Socket.IO for persistent real-time streaming connections.

---
